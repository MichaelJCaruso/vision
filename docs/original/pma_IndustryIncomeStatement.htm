Application: IndustryIncomeStatement

 

Summary:

The IndustryIncomeStatement application is used to display current income statement information for the largest companies in an industry based on total sales.

Inputs:

Position Form Id Description 
1  id  Id of the industry 

Analysis:

  • The id must reference an existing Industry instance.
  • The basic report displays the income statement line items as rows and the top 5 companies in the industry (by sales) as columns.

Key Data Feeds:

  • FundamentalA
  • CompanyToIndustry

Related Reports:

  • IndustryProfile
  • CompanyIncomeStatement
  • IndustryBalanceSheet

Sample Usage:

     Interface ApplicationWS runWith: "IndustryIncomeStatement", "530" ;
