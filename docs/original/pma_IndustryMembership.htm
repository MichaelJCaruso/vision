Application: IndustryMembership

 

Summary:

The IndustryMembership application is used to display the companies comprising an industry at a particular point in time and the companies added and dropped from the industry on the last date it was changed.

Inputs:

Position Form Id Description 
1  id  Id of the industry 
2  -  Global date override 

Analysis:

  • The id must reference an existing Industry instance.
  • The basic report displays the list of companies in the industry followed by the companies added to the industry on the last date the industry membership was updated and the companies dropped from the industry on this date.

Key Data Feeds:

  • CompanyToIndustry

Related Reports:

  • IndustryProfile

Sample Usage:

    Interface ApplicationWS runWith: "IndustryMembership", "530" ;
