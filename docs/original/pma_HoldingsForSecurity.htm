Application: HoldingsForSecurity

 

Summary:

The HoldingsForSecurity application is used to display the portfolios that hold a security as of a date.

Inputs:

Position Form Id Description 
1  id  Id of the security 
2  -  global date override 
3  -  Sorting criteria 

Analysis:

  • The id must reference an existing Security instance.
  • If parameter 3 is supplied, it specifies the sorting criteria. By default, the report is sorted from highest to lowest shares.
  • This report is restricted to Portfolios that own the security. No other Account subclasses are included.
  • The basic report displays portfolios as rows and the following columns:
    • Portfolio Id
    Portfolio Name
    Shares held in Portfolio
    Total market value of holding
    Percent of portfolio in this security
  • If no global currency has been specified, the portfolio market values are expressed in the security's base currency.

Key Data Feeds:

  • HoldingsFeed

Related Reports:

  • SecurityProfile
  • AccountProfile
  • HoldingsHistoryForSecurity

    Interface ApplicationWS runWith: "HoldingsForSecurity", "DELL" ;
