Application: AccountHoldingsHistory

 

Summary:

The AccountHoldingsHistory application is used to display the dates for which holdings have been retrieved or derived for an account. The total market value and the market value summarized by asset type is displayed for each date.

Inputs:

Position Form Id Description 
1  id  Id of the account 

Analysis:

  • The id must reference an existing Account instance.
  • The basic report displays dates as rows and the following columns:
    • Date
    Number of holdings on date
    Total market value
    Market value of Cash assets
    Market value of Equity assets
    Market value of Fixed Income assets

Key Data Feeds:

  • HoldingsFeed

Related Reports:

  • AccountProfile
  • AccountHoldings

Sample Usage:

    Interface ApplicationWS runWith: "AccountHoldingsHistory", "102" ;
