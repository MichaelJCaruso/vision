Application: AccountBreakdown

 

Summary:

AccountBreakdown application is used to summarize the holdings in an account based on a user-specified grouping criteria. The total market value, group weight, and number of holdings in group are displayed in the summary version of this report. A detail option allows you to see the individual holdings for a specific group or for all groups.

Inputs:

Position Form Id Description 
1  id  Id of the account 
2  group  Grouping criteria (from menu) 
-  groupEntry  Override for group value (from input) 
3  -  Specific group to display 
-  details  Indicates if all details should be displayed 

Analysis:

  • The id must reference an existing Account instance.
  • If parameter 2 or the group field is supplied, it specifies the grouping criteria. If the groupEntry field is also supplied, it will be used as the criteria. The default grouping criteria is security company sector.
  • If parameter 3 is supplied, it indicates that details should be displayed for this group. If the details field contains the string Detail, details will be displayed for all groups.
  • The basic report displays groups as rows and the following columns:
    • Group Id
    Group Name
    Total Market Value of the group
    Total weight in the group
    Number of holdings in group
    If a specific group or "all" details is requested, the actual securities held in the group and their market values, percentage, and shares are also displayed.
  • If this is a summary level report, a pie chart is displayed showing the percentage held in the largest 9 groups.
