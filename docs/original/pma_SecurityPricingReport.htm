Application: SecurityPricingReport

 

Summary:

The SecurityPricingReport application is used to display history prices for a security.

Inputs:

Position Form Id Description 
1  id  Id of the security 

Analysis:

  • The id must reference an existing Security instance.
  • If the current currency is US dollars, the last 10 days and the last 25 month-ends of closing price and pe ratio are displayed. A monthly price chart is also created.
  • If the current currency is not US dollars, the last 25 month-ends of price in local currency, price in US dollars, and exchange rate data are displayed.

Key Data Feeds:

  • PriceFeed
  • FundamentalM

Related Reports:

  • SecurityProfile

Sample Usage:

    Interface ApplicationWS runWith: "SecurityPricingReport", "DELL" ;
