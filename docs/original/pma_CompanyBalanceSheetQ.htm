Application: CompanyBalanceSheetQ

 

Summary:

The CompanyBalanceSheetQ application is used to display the last five fiscal quarters of balance sheet information for a company.

Inputs:

Position Form Id Description 
1  id  Id of the company 

Analysis:

  • The id must reference an existing Company instance.
  • The basic report displays the balance sheet line items as rows and the last five fiscal quarter end dates as columns.

Key Data Feeds:

  • FundamentalQ

Related Reports:

  • CompanyBalanceSheet
  • CompanyIncomeStatementQ
  • CompanyProfile

Sample Usage:

    Interface ApplicationWS runWith: "CompanyBalanceSheetQ", "DELL" ;
