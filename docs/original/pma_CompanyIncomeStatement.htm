Application: CompanyIncomeStatement

 

Summary:

The CompanyIncomeStatement application is used to display the last three years of income statement information for a company.

Inputs:

Position Form Id Description 
1  id  Id of the company 

Analysis:

  • The id must reference an existing Company instance.
  • The basic report displays the income statement line items as rows and the last three fiscal year end dates as columns.

Key Data Feeds:

  • FundamentalA

Related Reports:

  • CompanyIncomeStatementQ
  • CompanyBalanceSheet
  • CompanyProfile

Sample Usage:

    Interface ApplicationWS runWith: "CompanyIncomeStatement", "DELL" ;
