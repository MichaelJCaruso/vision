Application: UniverseMembership

 

Summary:

The UniverseMembership application is used to display the entities comprising a universe at a particular point in time and the entities added and dropped from the universe on the last date it was changed.

Inputs:

Position Form Id Description 
1  id  Id of the universe 
2  -  Global date override 

Analysis:

  • The id must reference an existing Universe instance.
  • The basic report displays the list of entities in the universe followed by the entities added to the universe on the last date the membership was updated and the entities dropped from the universe on this date.

Key Data Feeds:

  • UniverseMembers

Related Reports:

  • UniverseProfile

Sample Usage:

    Interface ApplicationWS runWith: "UniverseMembership", "SP500" ;
