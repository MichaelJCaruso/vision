Application: CompanyProfile

 

Summary:

The CompanyProfile application is used to provide summary information about a company with links to more in depth information. The summary includes basic information about the company, financial highlights, and the list of securities issued by the company.

Inputs:

Position Form Id Description 
1  id  Id of the company 
2  -  Id of currency override 

Analysis:

  • The id must reference an existing Company instance.
  • The display includes four sections:
    • 1. Basics - basic information about the company including alternate ids
    2. Relationships - basic classifications of the company including country, industry, sector, and universe memberships
    3. Financial Highlights - latest fiscal dates and data
    4. Securities - list of securities issued by company

Key Data Feeds:

  • CompanyMaster
  • SecurityMaster
  • FundamentalA
  • FundamentalQ
  • FundamentalM

Related Reports:

  • SecurityProfile

     Interface ApplicationWS runWith: "CompanyProfile", "DELL" ;
